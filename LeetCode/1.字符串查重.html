实现一个算法，确定一个字符串 s 的所有字符是否全都不同。
        示例 1：
        输入: s = "leetcode"
        输出: false 
        
        提示：
        方法一：双重循环
        方法二：数组set方法去重
        方法二提示：Set和Map类似，也是一组key的集合，但不存储value。
        由于key不能重复，所以，在Set中，没有重复的key。
    
        split 字符串方法，splice、slice 数组方法 splice改变原数组，slice不改变原数组
        1、split()  返回一个字符串数组。
        split('以这个为分隔符',可指定返回的数组的最大长度)
        var str="How are you doing today?";
        var n=str.split(" ",3);
        n 将输出3个数组的值:
        How,are,you

        2、splice() 返回被删除的元素组成的数组 
        即 如果使用arr = arr.splice()的方法将会返回一个被截取的部分，
        而arr.splice()后再调用arr将会返回一个截取后的新数组。
        https://www.w3school.com.cn/jsref/jsref_splice.asp
        splice(从第几个开始|要删除的项目数量。如果设置为 0，则不会删除项目|增加的项)
        let arr = ['zz','213','x3','阿萨德','read']
        arr.splice(2,0,'abc')
        //返回[]
        // arr (6) ["zz", "213", "abc", "x3", "阿萨德", "read"]

        arr.splice(1,3,'456')
        //返回 ["213", "abc", "x3"]
        // arr (4) ["zz", "456", "阿萨德", "read"]

        3、slice() 返回一个新的数组，
        包含从 start 到 end （不包括该元素）的 arrayObject 中的元素。
        ['a','b','c','4'].slice(1,3)
        //返回 (2) ["b", "c"]
