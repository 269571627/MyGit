（1）值类型引用类型深入分析：
值类型：栈存储（从上往下存储）
key    value
a      100     ----> a = 200
b      100     （b = a）


引用类型：堆存储（从下往上存储）及栈存储，赋值的是内存地址
     栈
key     value
c       内存地址1       
d       内存地址1   （d = c）
  ........

内存地址1   {age：20}  ----->d.age:21
key        value
        堆

（2）常用值类型
定义为undefined、数字、字符串、布尔型、Symbol('s')

（3）常用引用类型
对象、数组、null（特殊的引用类型，指针指向为空地址）、函数（特殊的引用类型）
